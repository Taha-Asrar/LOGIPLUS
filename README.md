# LogiPlus – Spatial Database Project (PostGIS)

## Description
Ce projet implémente une base de données logistique avec PostgreSQL/PostGIS
et un tableau de bord web permettant de visualiser les analyses spatiales :
- appariement commandes ↔ entrepôts
- calculs de distance
- analytics produits et pays
- visualisation cartographique (Leaflet)

## Technologies
- PostgreSQL ≥ 14
- PostGIS
- PHP 8
- Leaflet.js
- SQL pur + PostGIS
